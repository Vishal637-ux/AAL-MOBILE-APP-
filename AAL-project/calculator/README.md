# Calculator

A simple browser calculator built with HTML, CSS, and JavaScript.

## Files

- `index.html` contains the calculator layout.
- `style.css` handles the visual design and responsive layout.
- `script.js` powers button clicks, keyboard input, deletion, clearing, and calculations.

## Run

Open `index.html` in a browser.

## Keyboard Shortcuts

- Numbers and operators: enter values
- `Enter` or `=`: calculate
- `Backspace`: delete
- `Escape`: clear
